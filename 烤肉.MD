## 遊戲概念

你是一塊食材，被扔上了滾燙的烤肉盤。
你不想被烤熟，你不想被推下去，你要活下去。

**核心目標**：在烤肉盤上撐過倒數計時，不被烤熟、不被推落、不被叉子刺中。

---

## 死亡條件

| 死亡方式 | 說明 |
|---------|------|
| 被烤熟 | 熟度條達到 100%，GG |
| 墜落 | 被推出烤肉盤邊緣，掉落即死 |
| 被叉子刺中 | 廚師叉子直接命中，立即死亡 |

## 推擠系統

### 基本操作
- **按鍵**：`E`（鍵盤）/ 右搖桿按下 `R3`（手把）/ 螢幕上獨立推擠鈕（手機）
- **方向**：朝玩家當前**移動方向**推出；靜止時朝**面向方向**推
- **冷卻**：2 秒，技能條 UI 獨立顯示

### 推擠機制

| 情況 | 結果 |
|------|------|
| 推到其他食材 | 對方被擊飛（距離依雙方重量差決定） |
| 推到烤盤邊緣的食材 | 極大機率直接推落，**立即死亡** |
| 推到火焰區的食材 | 對方被推入高溫格，熟度額外 +15% |
| 推到叉子路徑上的食材 | 對方被叉子刺中，立即死亡 |
| 推空（無目標） | 自身短暫向前滑行 0.5 格，可用於緊急閃避 |

### 重量影響
不同食材的推力與抗推力不同：

| 食材 | 推力倍率 | 被推飛距離 |
|------|---------|-----------|
| 五花肉 | 1.5× | 短（抗推） |
| 牛小排 | 1.3× | 短（抗推） |
| 培根 | 1.0× | 中 |
| 玉米 | 1.0× | 中 |
| 雞翅 | 0.8× | 長（輕盈） |
| 青椒 | 0.8× | 長 |
| 香菇 | 0.7× | 非常長 |
| 豆腐 | 0.6× | 非常長（易碎） |
| 秋刀魚 | 1.2× | 中（長軸方向加成） |
| 蝦子 | 0.7× | 長 |

> **豆腐特例**：被推時除了飛出，還需擲骰決定是否碎裂（機率 35%）

### 邊緣危險區
```
┌──────────────────────────┐
│  ████ 危險邊緣 ████      │  ← 站在此區被推 = 必落
│  ┌────────────────────┐  │
│  │  安全區             │  │
│  │                    │  │
│  └────────────────────┘  │
└──────────────────────────┘
```
- 站在危險邊緣時，畫面邊框閃紅色警示
- 靠近邊緣的食材推力抵抗降低 50%（腳踩空感）

### 食材特殊技能與推擠的互動

| 食材 | 互動說明 |
|------|---------|
| 五花肉 · 滑油衝刺 | 衝刺中撞到食材，直接觸發推擠效果（不消耗推擠 CD） |
| 香菇 · 縮水跳躍 | 被推瞬間可取消位移，觸發跳躍逃脫（消耗跳躍 CD） |
| 玉米 · 爆米花 | 被推進高溫區時若熟度 ≥ 70%，直接引爆，炸飛推他的人 |
| 秋刀魚 · 刀型衝刺 | 衝刺時推力 ×3，沿直線連續撞飛多個食材 |
| 豆腐 · 吸熱護盾 | 護盾期間被推，推力減半且不碎裂 |

## 熟度系統

- 站在火焰上方會持續累積熟度
- 熟度條：`[生] ░░░░░░░░░░ [熟透]`
- 移動到烤盤邊緣溫度較低，熟度累積變慢
- 部分食材可以主動降低熟度（見特殊能力）

## 可選食材（角色）

### 五花肉
- 生命值：高
- 移動速度：慢
- 特殊能力：**滑油衝刺** — 釋放豬油，在烤盤上滑行並推開周圍食材，也可以拿來躲叉子
- 弱點：油脂讓火焰傷害 +20%

### 牛小排
- 生命值：中
- 移動速度：中
- 特殊能力：**厚實裝甲** — 短暫時間內減少熟度累積速度 50%
- 弱點：體型大，難以閃避叉子

### 香菇
- 生命值：低
- 移動速度：快
- 特殊能力：**縮水跳躍** — 受熱時快速縮小並跳到對角位置
- 弱點：熟度累積比其他食材快 30%

### 玉米
- 生命值：中
- 移動速度：中
- 特殊能力：**爆米花爆炸** — 熟度超過 70% 時可引爆，噴飛周圍威脅（叉子、其他食材）
- 弱點：熟度超過 70% 後開始不穩定，有機率自爆

### 雞翅
- 生命值：低
- 移動速度：非常快
- 特殊能力：**短距滑翔** — 可以飛越火焰區域落到另一側
- 弱點：生命值最低，叉子差點打中也有傷害

### 蝦子
- 生命值：低
- 移動速度：快
- 特殊能力：**彎身彈射** — 蓄力後沿弧線高速彈射到指定位置，可越過障礙與火焰區
- 弱點：熟度超過 60% 後身體捲曲，移動速度下降 30%

### 豆腐
- 生命值：中
- 移動速度：慢
- 特殊能力：**吸熱護盾** — 短暫吸收周圍食材的熟度值，自身熟度上限提升至 130%（超過才死）
- 弱點：被撞擊時有機率直接碎裂，立即死亡

### 培根
- 生命值：低
- 移動速度：中
- 特殊能力：**捲縮甩打** — 受熱後捲曲成陀螺狀高速旋轉，撞飛周圍所有食材
- 弱點：貼合烤盤面積大，熟度累積速度 +40%

### 青椒
- 生命值：中
- 移動速度：中
- 特殊能力：**噴辣氣** — 向四周噴出辣氣，使接觸的食材短暫減速並附加熟度 +10%
- 弱點：不耐高溫，自身熟度累積速度 +25%

### 秋刀魚
- 生命值：中
- 移動速度：中（橫向移動較快）
- 特殊能力：**刀型衝刺** — 沿身體長軸方向高速衝刺，沿路撞飛所有食材與叉子
- 弱點：體型細長，轉向遲緩，極難閃避正面叉子攻擊

## 場地設計

```
┌──────────────────────────────────────┐
│  韓式烤肉餐廳背景（木質桌面、暖燈）    │
│  ┌────────────────────────────────┐  │
│  │  烤盤（鑄鐵烤網格條）            │  │
│  │  🔥🔥🔥  💀  🔥🔥🔥           │  │
│  │  🥩  🍄 [道具] 🌽             │  │
│  │  🔥🔥     🔥🔥🔥             │  │

└──────────────────────────────────────┘
```

> 火焰數量：8 個獨立火點，各自移動、振盪  
> 警告機制：淡黃預警 1.0 秒 → 火紅正式點燃 1.4–2.8 秒 → 移往下個位置

### 場地事件（隨機觸發）

| 事件 | 說明 | 警示時間 |
|------|------|---------|
| 火焰噴發 | 8 個火點獨立移動，強迫玩家持續走位 | 1.0 秒淡黃閃爍警告 |
| 廚師叉子 | 九宮格範圍（3×3 格）落下，配廚師登場動畫 | 黃色格線預警 1.5 秒 |
| 廚師翻面 | 整個烤盤傾斜，所有食材往同方向滑 | 廚師手伸入警示 |
| 刷醬攻擊 | 大刷子掃過，增加熟度 +30% | 刷子影子 |
| 加炭 | 烤盤全區溫度瞬間上升，持續 5 秒 | 火焰變大動畫 |

## 道具系統

道具每 5–9 秒在烤盤上隨機出現，最多同時存在 3 個。踩上即拾取，12 秒後自動消失（消失前閃爍提示）。

| 道具 | 類型 | 效果 | 持續時間 |
|------|------|------|---------|
| ❄ 冰塊 | 好道具 | 立刻降低熟度 25% | 即時 |
| ◎ 鋁箔紙 | 好道具 | 火焰傷害完全隔絕 | 4 秒 |
| ★ 大蒜 | 好道具 | 推擠力道 ×2 | 5 秒 |
| ! 醬油 | 壞道具 | 熟度累積速度 ×2 | 5 秒 |
| ! 胡椒粉 | 壞道具 | 移動速度降低 50% | 3 秒 |

> 好道具邊框為藍綠色，壞道具邊框為紅色，一眼可分辨

## 多人模式

- 2 ~ 4 人同時在同一烤盤上競爭求生
- 最後一個存活的食材獲勝
- **互動機制**：
  - 可以推擠其他食材進入火焰區
  - 可以利用自己的特殊能力影響他人
  - 玉米爆炸可以炸飛所有人（包括自己）

## 關卡場景

### 關卡一：家庭烤肉架
- 火焰較規律，廚師只有叉子攻擊
- 難度：★☆☆☆☆

### 關卡二：餐廳鐵板燒
- 場地更小，溫度更高
- 廚師動作變快，增加刷醬攻擊
- 難度：★★★☆☆

### 關卡三：露營野火烤架
- 風吹使火焰方向不固定
- 烤架會晃動（傾斜更頻繁）

### 關卡四：地獄料理節目直播
- 兩個廚師同時攻擊
- 計時器縮短
- 觀眾投票觸發隨機災難
- 難度：★★★★★
