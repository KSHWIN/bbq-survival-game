### 各角色詳細說明

**五花肉** — 力量最強，體型最大，速度最慢。招式「滑油衝刺」在烤盤留油漬讓對手失控；大絕在原地向四方狂噴豬油，無敵5秒。弱點：火焰傷害 +20%。

**牛小排** — 體型壯、抗推。招式「厚實裝甲」短暫減少熟度累積；大絕「鐵板衝鋒」蓄力3秒後自動衝鋒，撞到敵人往旁邊強力彈飛。

**香菇** — 移動快，但熟度累積速度最快。招式「縮水跳躍」受熱時瞬移到安全位置；大絕「孢子迷霧」持續噴出霧氣，久待其中的敵人定身。

**玉米** — 均衡型。招式「爆米花」熟度 ≥ 70% 時可引爆炸飛周圍；大絕「爆米花連炸」亂射爆米花，射中敵人動彈不得。

**雞翅** — 速度最快，但體型最輕、最容易被推飛。招式「短距滑翔」可越過火焰降落另一側；大絕「旋風疾飛」爆速移動5秒並瞬移到瞄準終點。

**蝦子** — 輕盈快速。招式「彎身彈射」蓄力後弧線高速彈到指定位置；大絕「衝擊浪潮」蓄力3秒可調整方向，釋放寬幅短程海浪，中央推力最強。

**豆腐** — 速度最慢，體型最小，免疫自身踩到的油漬。招式「吸熱護盾」吸收周圍敵人熱度；大絕「冷卻護盾」瞬間降低熟度並展開護盾。被撞擊有機率碎裂死亡。

**培根** — 熟度累積極快。招式「捲縮甩打」旋轉陀螺甩飛周圍；大絕「煙燻覆蓋」全場鋪滿煙霧，讓所有敵人減速9秒。

**青椒** — 大絕最有追逃感。招式「噴辣氣」使接觸者減速；大絕「辣椒素噴霧」速度+50%，跑過的路線留下辣椒素，踩到的敵人減速75%達3秒。

**秋刀魚** — 體型細長，衝刺力強。招式「刀型衝刺」沿長軸連續撞飛多個目標；大絕「魚骨連射」超高速彈珠模式持續8秒在場內亂彈。

## 遊戲概念

你是一塊食材，被扔上了滾燙的烤肉盤。
你不想被烤熟，你不想被推下去，你要活下去。

**核心目標**：在烤肉盤上撐過倒數計時，不被烤熟、不被推落、不被叉子刺中。

## 死亡條件

| 死亡方式 | 說明 |
|---------|------|
| 被烤熟 | 熟度條達到 100%，GG |
| 墜落 | 被推出烤肉盤邊緣，掉落即死 |
| 被叉子刺中 | 廚師叉子直接命中，立即死亡 |

## 推擠系統

### 基本操作
- **按鍵**：`E`（鍵盤）/ 右搖桿按下 `R3`（手把）/ 螢幕上獨立推擠鈕（手機）
- **方向**：朝玩家當前**移動方向**推出；靜止時朝**面向方向**推
- **冷卻**：2 秒，技能條 UI 獨立顯示

### 推擠機制

| 情況 | 結果 |
|------|------|
| 推到其他食材 | 對方被擊飛（距離依雙方重量差決定） |
| 推到烤盤邊緣的食材 | 極大機率直接推落，**立即死亡** |
| 推到火焰區的食材 | 對方被推入高溫格，熟度額外 +15% |
| 推到叉子路徑上的食材 | 對方被叉子刺中，立即死亡 |
| 推空（無目標） | 自身短暫向前滑行 0.5 格，可用於緊急閃避 |

### 重量影響
不同食材的推力與抗推力不同：

| 食材 | 推力倍率 | 被推飛距離 |
|------|---------|-----------|
| 五花肉 | 1.5× | 短（抗推） |
| 牛小排 | 1.3× | 短（抗推） |
| 培根 | 1.0× | 中 |
| 玉米 | 1.0× | 中 |
| 雞翅 | 0.8× | 長（輕盈） |
| 青椒 | 0.8× | 長 |
| 香菇 | 0.7× | 非常長 |
| 豆腐 | 0.6× | 非常長（易碎） |
| 秋刀魚 | 1.2× | 中（長軸方向加成） |
| 蝦子 | 0.7× | 長 |

> **豆腐特例**：被推時除了飛出，還需擲骰決定是否碎裂（機率 35%）

### 邊緣危險區

┌──────────────────────────┐

│  ┌────────────────────┐  │

- 靠近邊緣的食材推力抵抗降低 50%（腳踩空感）

### 食材特殊技能與推擠的互動

| 食材 | 互動說明 |
|------|---------|
| 五花肉 · 滑油衝刺 | 衝刺中撞到食材，直接觸發推擠效果（不消耗推擠 CD） |
| 香菇 · 縮水跳躍 | 被推瞬間可取消位移，觸發跳躍逃脫（消耗跳躍 CD） |
| 玉米 · 爆米花 | 被推進高溫區時若熟度 ≥ 70%，直接引爆，炸飛推他的人 |
| 秋刀魚 · 刀型衝刺 | 衝刺時推力 ×3，沿直線連續撞飛多個食材 |
| 豆腐 · 吸熱護盾 | 護盾期間被推，推力減半且不碎裂 |

## 熟度系統

- 站在火焰上方會持續累積熟度
- 熟度條：`[生] ░░░░░░░░░░ [熟透]`
- 移動到烤盤邊緣溫度較低，熟度累積變慢
- 部分食材可以主動降低熟度（見特殊能力）

## 道具系統

道具每 5–9 秒在烤盤上隨機出現，最多同時存在 3 個。踩上即拾取，12 秒後自動消失（消失前閃爍提示）。

| 道具 | 類型 | 效果 | 持續時間 |
|------|------|------|---------|
| ❄ 冰塊 | 好道具 | 立刻降低熟度 25% | 即時 |
| ◎ 鋁箔紙 | 好道具 | 火焰傷害完全隔絕 | 4 秒 |
| ★ 大蒜 | 好道具 | 推擠力道 ×2 | 5 秒 |
| ! 醬油 | 壞道具 | 熟度累積速度 ×2 | 5 秒 |
| ! 胡椒粉 | 壞道具 | 移動速度降低 50% | 3 秒 |

> 好道具邊框為藍綠色，壞道具邊框為紅色，一眼可分辨

### 大絕能量球

場上會定期出現**發光能量球**，出現前 3 秒地板會有紫色光暈預告，出現後有粒子環繞旋轉。收集 3 顆即可按 **D** 釋放大絕。大家都搶著撿，有互動感。

## 多人模式

- 2 ~ 4 人同時在同一烤盤上競爭求生
- 最後一個存活的食材獲勝
- **互動機制**：
  - 可以推擠其他食材進入火焰區
  - 可以利用自己的特殊能力影響他人
  - 玉米爆炸可以炸飛所有人（包括自己）

## 關卡場景

### 關卡一：家庭烤肉架
- 火焰較規律，廚師只有叉子攻擊
- 難度：★☆☆☆☆

### 關卡二：餐廳鐵板燒
- 場地更小，溫度更高
- 廚師動作變快，增加刷醬攻擊
- 難度：★★★☆☆

### 關卡三：露營野火烤架
- 風吹使火焰方向不固定
- 烤架會晃動（傾斜更頻繁）
- 難度：★★★★☆

### 關卡四：地獄料理節目直播
- 兩個廚師同時攻擊
- 計時器縮短
- 觀眾投票觸發隨機災難
- 難度：★★★★★
